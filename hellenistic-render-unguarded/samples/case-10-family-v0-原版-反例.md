## 家庭

**范围提醒**: 本盘伦理护栏标准开 for 真实人物——**子女话题关、兄弟姐妹话题关**。本章只覆盖**父母 / 血脉 / 根基**表册，不展开子女 / 兄弟姐妹作为话题。

**核心特质**: 案主的家庭 / 起源**走"根基管理者在自我门口最难入境 + 父母在整盘最紧对冲直接互见 + 母近父远轴距 + 隐位 8 主土星承担血脉负担"四层叠加**。所以**适合 / 顺势能做的**：接受家是"结构性扛起来的承担"而不强求温情共生；以承担一部分但不独自全扛的方式跟家保持节奏；接受母性那层"贴在自我门口、跟内感与驱动捆在一起"的事实；接受父性那层走"从对面 / 一段距离照过来"的间接通道。

家基这条线走 4 室空 + 4 主火星在 1 室最难入境——根基轴管理者**塌缩到自我门口最难的一颗行星**，并且同时管 IC + 9 室 + 父点 + 子女点（**家庭看法三汇承担**：一颗行星同时承担 3 个家庭根基本性主角色 + 1 个交叉点角色）。读成"破裂 / 不稳定家庭基础"会漏入境主光接纳 + 木星三合缓冲两条通道，也漏"4 主在 1 室 = 根基经由自我锚过来"的结构事实。

父母协调走日月紧对（1°32' 渐合对冲，近满月）+ 跨层互纳容纳——父本性主（太阳）+ 母本性主（月亮）在整盘最紧的对冲带跨层互相容纳，近满月又给到最大光照。**父母线不是分离的领域**，坐在同一条轴上直接相位 + 互相住对方家。但**结构距离不同**：母亲影响**结构性贴近门口**（月亮在 1 室自轴 + 跟火星紧合在同一空间，母性影响和内感 / 动作高度交织）；父亲影响**结构性跨轴**（太阳在 7 室 + 与火星 4°37' 跨轴对冲），通过对面 / 经由遇境过来。

血脉那条走土星（长辈 / 血脉本性主）从 8 室——**血脉负担嵌入整盘最分布的结构中轴**。母点在 8 室被木星管，木星在 5 室（远），母通道**结构性分裂为近（月在 1）+ 远（点在 8 隐位）**。父点的公式触发了次公式（PDF Jup−Mars 因为 |Sat−Sun|=16°31'<17° 触发 Schoener / Bonatti 次公式条件），仅供参考记录。

**不适合的**:
- 稳定 / 自由流动 / 温情避风港型家庭根基框架
- 单通道父母协调框架（只读"父主导"或"母主导"）
- 跟家走"直接说清楚 + 频繁深度对话"型
- 把家拉进创造 / 美感层共生（不同层，强求会冲突）
- **绝不适合 父母身体 / 健康 / 死亡 / 寿命的预测断言**——伦理护栏硬底线，per `fictional-subject-ethical-guard.md` 真实人物标准
- 把根基当成跟自我 / 职业 / 伴侣分离的领域
- 平均交点作为祖先解读的锚（仅作源记录政策）

**这盘的家是结构性扛起来的家，不是温情包裹的家**——不是冷淡或疏离，是**结构事实**而非情感缺失。
