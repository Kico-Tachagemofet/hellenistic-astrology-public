# 渲染 checklist（每段过一遍）

> 本文件是 SKILL.md workflow 的 Step 2 子步骤. 如果你没有走完 Step 1 + Step 1.5,
> 先回去走——没有生活判断摘要就开写, checklist 救不了你.

## 前置 gate（不可跳过）

- **只读 topic-findings**, 不读 composition.md / chart-finding-index.md
- **Step 1.5 mandatory**: 写叙事前先输出生活判断摘要——每个 TF 列出所有可验证判断点（不是一句话, 是 bullet list）, 每条读者能直接打勾打叉
- **写完每段后过 anti-cliche-astrology.md E/F/G/H/I 做文体 self-review**（不在此处, 在写后）

---

## 叙事段

1. 这句话一个不懂占星的人能说"准"或"不准"吗？不能 → 重写
2. 有没有 chart-tech inline？（行星名 / 宫位号 / 度数 / 尊贵状态 / 相位名）→ 移到 trailing
3. 有没有"不是 X 而是 Y"句式？→ 删掉，直接说 Y
4. 有没有排比 / 对仗 / 列举？→ 改成散文
5. 有没有"结构性 XX"/ "XX 模式"/ "XX 通道"/ "走 X 线"？→ 改成具体场景
6. 有没有解释尾巴（"这意味着""这显示出"）？→ 删掉
7. 有没有同一件事换说法重复？→ 只留一种说法
8. 时间节点有没有括号写"为什么"？→ 没有就补
9. 收尾超过一句话了吗？→ 砍到一句
10. 小标题是 reader-facing 的吗？→ 不是就改（"跟父亲"不是"Sun 7H"）
11. 每句话是一个可以单独打勾或打叉的判断吗？→ 如果是对一种感觉的描写（"像背景音乐""自带灼热感"），改成具体可验证的判断（"要认出自己在感受什么通常得靠旁人点一句"）→ 描写可以有，但必须跟在判断后面作补充，不能替代判断

## trailing 技术依据

- chart facts 从 topic-findings 的 audit refs 取
- 术语用第一层固定译名（per terminology-register.md）
- cross-topic anchor 行 optional（F-id 从 topic-findings 的"Lifts to"行取，不回查 chart-finding-index）

## 禁词速查

叙事段不能出现: baseline / backdrop / backbone / carrier / peak / routing / channel / bridge / anchor / load / stack / lock / window / track / split / bifurcation / concurrent / 结构性XX（单用不续场景）/ XX模式 / XX通道 / 走X线

## 中文母语速查

- 动词长骨肉（"停"→"停下来"）
- 主题开场不主语开场（"这件事, 他想了一整夜"）
- 主语能省就省
- 写完反翻英语——结构不变说明是翻译腔
